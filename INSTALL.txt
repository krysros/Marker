Marker
======

Requirements
------------

- Python 3 >= 3.10
- PostgreSQL

Create database
---------------

- Run psql.

    sudo -u postgres psql

- If you are using Windows

    psql -U postgres

- Create database.

    create database marker;

- Create testing database (optional; only for pytest).

    create database testing_marker;

- Quit psql.

    \q

Configuration
-------------

- Edit configuration files: development.ini, testing.ini, production.ini.
  The typical form of a database URL is:

    dialect+driver://username:password@host:port/database

Getting Started
---------------

- Change directory into your newly created project if not already there. Your
  current directory should be the same as this INSTALL.txt file and setup.py.

    cd marker

- Create a Python virtual environment, if not already created.

    python3 -m venv env

- Upgrade packaging tools, if necessary.

    env/bin/pip install --upgrade pip setuptools wheel

- Install the project in editable mode with its testing requirements.

    env/bin/pip install -e ".[testing]"

- Initialize and upgrade the database using Alembic.

    env/bin/alembic -c development.ini upgrade head

- Load default data into the database using a script.

    env/bin/initialize_marker_db development.ini

- Run your project's tests.

    env/bin/pytest

- Run your project.

    env/bin/pserve development.ini

- Log in as admin using default password: admin

- Set up an admin account and create user accounts.

Internationalization
--------------------

- Run these commands in project's directory

    python setup.py extract_messages -o marker/locale/messages.pot

    python setup.py init_catalog -l pl -i marker/locale/messages.pot -o marker/locale/pl/LC_MESSAGES/messages.po

    python setup.py update_catalog -l pl -i marker/locale/messages.pot -o marker/locale/pl/LC_MESSAGES/messages.po

    python setup.py compile_catalog -i marker/locale/pl/LC_MESSAGES/messages.po -o marker/locale/pl/LC_MESSAGES/messages.mo
